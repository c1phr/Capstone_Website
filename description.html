Project Information

This mobile application is for users to form into virtual groups with the intention of keeping track of one another via geolocation as well as messaging. Groups are easy to start either via Wifi Direct or a standard email/sms invite, and groups would exist for short periods of time, dissolving when all users are finished with their activity.

The Lightweight Groups project involved the creation of a proof of concept application on the Android platform called Toogether.
