Project Information

Prototype 1: The first version of this prototype will contain a gps function that finds the user of the application as well as other people that are considered in a group. It will also contain a grouping function to create a collective amount of information so as to communicate with the other people in the group through a chat. This means that there will also be a chat room function for text based chat purposes.
